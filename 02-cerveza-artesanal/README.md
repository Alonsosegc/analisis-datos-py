# 🍺 Análisis de Cerveza Artesanal

Análisis estadístico del porcentaje de alcohol en una línea de producción 
de cerveza artesanal utilizando NumPy. Se aplican técnicas de control de 
calidad para detectar valores fuera de los límites permitidos.

## ¿Qué hace este proyecto?

- Carga de datos desde archivo CSV con NumPy
- Cálculo de media y desviación estándar
- Determinación de límites de control (LIC y LSC)
- Identificación de lotes fuera de los límites de control

## Resultados
- Media: 5.010%
- Desviación estándar: 0.312%
- Límite inferior: 4.076%
- Límite superior: 5.945%
- Valores fuera de límites: [3.87, 6.02]

## Tecnologías
- Python 3
- NumPy
- Jupyter Notebook
